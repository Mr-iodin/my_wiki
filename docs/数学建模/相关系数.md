# 相关系数

### 1. Person相关系数

用来衡量两个变量之间相关性大小指标。

总体——要考察对象全部个体；样本——从总体中抽取一部分个体；

##### 1.1 总体Person相关系数

两组数据X:{X<sub>1</sub>,X<sub>2</sub>,X<sub>3</sub>,...,X<sub>n</sub>}和Y:{Y<sub>1</sub>,Y<sub>2</sub>,Y<sub>3</sub>,...,Y<sub>n</sub>}是总体数据，

总体均值：

![image-20220310120337114](https://s2.loli.net/2022/03/10/Ix1LR8zgecNQ9SA.png)

总体协方差：

![image-20220310120441711](https://s2.loli.net/2022/03/10/QqFIPmGB9sfinVT.png)

​	简单理解就是X、Y变化方向一直保持相同（均大于/小于均值），则协方差为正；X、Y变化方向一直相反，协方差为负；X、Y变化方向间相互无规律，那么分子中既有正又有负，那么在累加过程中正负会抵消。

所以在一定程度上，协方差可以反应X和Y的相关性，但协方差的大小和两个变量的量纲有关，因此不适合做相关性的比较。

而Person相关系数，就是通过对变量进行标准化在总体协方差基础上消去量纲影响。

总体Person相关系数：

![image-20220310120459025](https://s2.loli.net/2022/03/10/8vGNLCK4RnWMpac.png)

σ<sub>X</sub>(sigma X)是X的标准差：

![image-20220310120515651](https://s2.loli.net/2022/03/10/tQzlRnFS4xhKZUN.png)

可证明,

* |ρ<sub>XY</sub>|≤1 ;

* 当Y=aX+b时,既X和Y在一条直线上时

  ![image-20220310120529544](https://s2.loli.net/2022/03/10/7qxlcuI5Wsawng2.png)

所以皮尔逊反映的是一种**线性相关系数**。

**总结:**

* 若两个样本本身就是线性相关关系,那么皮尔逊相关系数绝对值越大,相关性越强,反之越弱;
* 在不确定两个变量是什么相关的情况下,则不能随便使用皮尔逊相关系数下结论;

##### 1.5 皮尔逊相关系数进行假设检验

**Ⅰ  假设检验步骤：**

①确定原假设H<sub>0</sub>和备择假设H<sub>1</sub>；

②在原假设H<sub>0</sub>成立的条件下，根据要检验的量构造一个分布（正态、t、F、卡方，在构造分布中涉及了很多证明比较复杂）；

③画出概率密度图；

④给一个置信水平β，即相信H<sub>0</sub>成立的概率，ρ（a≤统计量≤b）=β（β一般取90%、95%、99%），若β取95%、对应的α则为5%；

⑤用已知样本数据代入统计量，得到检验值，若检验落在[a,b]内，则无法拒绝原假设，反之拒绝；

**Ⅱ 皮尔逊相关系数进行假设检验：**

* 假设检验

*第一步：*当我们计算出一个皮尔逊相关系数r时，我们想检验它是否显著的异于0，那么设定的原假设和备择假设可以为：
$$
H_0:r=0 ,H_1:r≠0
$$
*第二步：*在原假设成立的条件下，利用我们要检验的量构造一个符合某一分布的统计量。

对于皮尔逊相关系数r而言，在<u>满足一定条件</u>下可以构造统计量，
$$
t=r\sqrt{\frac{n-2}{1-r^2}}
$$
可以证明t是服从自由度为n-2的t分布。

*第三步：*将我们要检验的这个值带入这个统计量中，可以得到一个特定的值（检验值）。

​	如果计算的相关系数为0.5，n=30，那么可以得到：
$$
t^*=0.5\sqrt{\frac{30-2}{1-0.5^2}}=3.05505
$$
*第四步：*由于我们知道统计量的分布情况，因此我们可以画出该分布的概率密度函数pdf，并给定一个置信水平，根据这个置信水平查表找到临界值，并画出检验统计量的接受域和拒绝域。

上述统计量服从自由度n-2=30-2=28的t分布，可画出概率密度函数图像：

```matlab
%% 假设检验部分
x = -4:0.1:4;
y = tpdf(x,28);  %求t分布的概率密度值 28是自由度  
figure(1)
plot(x,y,'-')
grid on  % 在画出的图上加上网格线
hold on  % 保留原来的图，以便继续在上面操作
```

![](https://i.loli.net/2021/02/01/KsNoakpyd3OwGHz.png)

确定置信水平：90%、95%、99%

因为要侧双侧水平，所以需要找到覆盖0.95概率部分，

通过查找t分布表，对应临界值为0.048，可以做出拒绝域

```python
%通过matlab函数查找p分布表
tinv(0.975,28)    %  2.0484
% 这个函数是累积密度函数cdf的反函数
plot([-2.048,-2.048],[0,tpdf(-2.048,28)],'r-')
plot([2.048,2.048],[0,tpdf(2.048,28)],'r-')
```

![](https://i.loli.net/2021/02/01/WjL8hNRwyJqMGvs.png)

*第五步：*查看计算出的检验值落在接受域还是拒绝域,并下结论。

由于得到的t<sup>*</sup>=3.05505>2.048,所以可以得到结论：

在95%水平的置信水平上，我们拒绝原假设H<sub>0</sub>:r=0,因此r是显著的不为0。

* P值检验法

```matlab
%% 计算p值
x = -4:0.1:4;
y = tpdf(x,28);
figure(2)
plot(x,y,'-')
grid on 
hold on
% 画线段的方法
plot([-3.055,-3.055],[0,tpdf(-3.055,28)],'r-')
plot([3.055,3.055],[0,tpdf(3.055,28)],'r-')
disp('该检验值对应的p值为：')
disp((1-tcdf(3.055,28))*2)  %双侧检验的p值要乘以2

% 该检验值对应的p值为：0.0049
```

![](https://i.loli.net/2021/02/01/iNdwVpbA7gjKu2S.png)

根据检验值t<sup>*</sup>=3.05505,可以计算对应的P值。

基本思想：计算单测检验p值→乘以2得到双侧检验p值。

* p值>0.01，说明在99%置信水平无法拒绝原假设；
* p值>0.05，说明在95%置信水平无法拒绝原假设；
* p值>0.10，说明在90%置信水平无法拒绝原假设；

在本例中，拒绝原假设意味着皮尔逊相关系数显著的异于0。

### 3.斯皮尔曼spearman相关系数

若没有通过相关性检验,则可以用

X和Y为两组数据,其斯皮尔曼相关性:
$$
r_s=1-\frac{6\sum_{i=1}^n{d_{i}^{2}}}{n\left( n^2-1 \right)}
$$
![image-20210201220231827](https://i.loli.net/2021/02/01/9glDivq1PwO548B.png)

一个数的等级,就是将它所在的一列数据按照从小到大排序后,这个数所在位置

![image-20210201220803392](https://i.loli.net/2021/02/01/ykPtSICYvTj2hea.png)

注:若数值相同,则它们所在位置取算数平均。

可证明：r<sub>s</sub>位于-1和1之间。



所以，斯皮尔曼相关系数被定义为等级之间的皮尔逊相关系数。


### 4.两个相关系数比较

斯皮尔曼相关系数和皮尔逊相关系数选择:

1.连续数据，正态分布，线性关系，用pearson相关系数是最恰当，当然用
spearman相关系数也可以， 就是效率没有pearson相关系数高。

2.上述任一条件不满足，就用spearman相关系数，不能用pearson相关系数。

3.两个定序数据之间也用spearman相关系数，不能用pearson相关系数。

* 定序数据是指仅仅反映观测对象等级、顺序关系的数据，是由定序尺度计量形成的，表现为类别，可以进行排序，属于品质数据。
* 例如：优、良、差；
  我们可以用1表示差、2表示良、3表示优，但请注意，用2除以1得出的2并不代表任何含义。定序数据最重要的意义代表了一组数据中的某种逻辑顺序。

4.注：斯皮尔曼相关系数的适用条件比皮尔逊相关系数要广，只要数据满足单调关系
（例如线性函数、指数函数、对数函数等）就能够使用。
