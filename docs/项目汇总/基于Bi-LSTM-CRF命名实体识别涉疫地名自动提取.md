# 详细方案（Bi-LSTM-CRF）

## 基于Bi-LSTM-CRF命名实体识别

贴出来检讨一下，

由于沟通原因导致前期所有结果作废，无奈选题的前3天才决定改题做的命名实体识别，

而队友都是偏数学理论专业，之前都没接触过NLP，导致接下来的建模编程以及论文部分基本上都是我一个人在瞎扯，整整19页论文都写着满满“业余”，

2天爬虫，2天选择模型并学习理论，2天用于编程跟模型训练，剩下3天完成了论文与数据的整理，长达2个月时间周期的比赛就硬生生压成了9天...这锅我背我背。

由于前几天连续了5个通宵，到了后期脑子基本已经是放弃思考，论文中存在一句话三处语病的问题，所以接下来看论文的时候你就可以在里边发现无数个错别字......总之我是不忍心再把论文看一遍了。

论文中存在很多逻辑与理论上不严谨不准确的地方，赛后想了想也觉得当时用的方法也不太好，这也导致了最后得到的数据四分之一都是脏的以及大量的遗漏，之后如果进行更深入的学习的话会再回头进行更正与改进。

虽然论文写得很垃圾，但从某种意义上来讲这也算是我第一篇以论文形式的产出。从改题到提交作品，能做到这一步是我没想到的（我队友也没想到）,好像自己也没想象中那么垃圾嘛哈哈哈....（试图安慰自己

最后感谢队友陪我度过了难熬的两个月，下次国赛再一起拿个国一。
